To-Do List

To Do List is a web application built with JavaScript and React JS that allows users to create and manage tasks. The app provides a clean and user-friendly interface for users to add, edit, and delete tasks, as well as mark them as completed. Features: - Create new tasks with a title and description - Edit existing tasks to modify their title and description - Delete tasks that are no longer needed - Mark tasks as completed to keep track of progress - Filter tasks by completion status - Sort tasks by due date, priority, or title The application uses local storage to store tasks, so users can close the app and return later without losing their progress. The user interface is designed to be responsive and work seamlessly across different devices. This project demonstrates proficiency in JavaScript and React JS, as well as knowledge of web development best practices, including responsive design and local storage.
